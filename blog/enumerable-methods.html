Enumerables: The Group By Method

Feb 22 2016

Any data container in Ruby that utilizes the #each method can then be set to use the full set of Enumberable methods. These methods are stored under the Enumerable module, where they can be accessed by arrays and hashes, as well as any other containers you may need or create. This way, the methods are only created once and can be accessed in a number of ways (truly, this is DRY).

One Enumerable is the #group_by method, shown here:

sample_array.group_by {|i| ..code block.. }

This method will take an array and create a hash that organizes the array's elements according to the results of the code block. For instance,

array = [1,2,3,4,5]
array.group_by {|i| i%2 }
#=> {0=>[2,4], 1=>[1,3,5]}

In this example, our code block can return one of two values - either 0, if i is evenly divisible by 2, or 1, if it isn't. Our #group_by method creates a new hash creates a new key for each value that the code block returns, and then sorts each array element according to which value the code block returned for it.

This is really useful for reorganizing your data to get a more detailed structure around a specific data point, and can be very valuable as your data gets more and more complex.
