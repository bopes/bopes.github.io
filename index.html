Enumerables: The Group By Method

Feb 22 2016

Any data container in Ruby that utilizes the #each method can then be set to use the full set of Enumberable methods. These methods are stored under the Enumerable module, where they can be accessed by arrays and hashes, as well as any other containers you may need or create. This way, the methods are only created once and can be accessed in a number of ways (truly, this is DRY)...
